5 HTML Tags

<p>
Defines a paragraph of text within the document.
<img>
Embeds an image into the web page.
<h1>
Represents the highest-level heading, used for primary titles or headings.
<ul>
Creates an unordered list, typically displayed with bullet points.
<input>
Specifies an input field where the user can enter data, commonly used within forms.
